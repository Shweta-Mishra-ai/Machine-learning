## Overview

This repository is a structured, continually growing collection of practical machine learning implementations. It spans supervised learning such as regression and classification, unsupervised learning such as clustering and dimensionality reduction, natural language processing, model evaluation and tuning, and the underlying math foundations like the sigmoid function, softmax function, and vector operations. Many algorithms are implemented both with Scikit-Learn and from scratch, so the reasoning behind each model is visible alongside the production style version. The code is organized into topic folders, each with its own README, so the project can be browsed and reused like a real, maintained codebase rather than a loose set of scripts.

## Topics Covered

Supervised learning: simple and multiple linear regression, polynomial regression, logistic regression, Naive Bayes, k-nearest neighbors, decision trees. Unsupervised learning: hierarchical clustering, k-means clustering, principal component analysis, dummy variable handling. Natural language processing: sentiment analysis, sarcasm detection, bag of words, count vectorization, word frequency analysis. Model evaluation and tuning: grid search, k-fold cross validation, min-max scaling, label encoding, train-test splitting, model persistence with pickle and joblib. Math foundations: sigmoid function, softmax function, dot product and vector operations implemented from scratch.

## Tech Stack

Python, Scikit-Learn, Pandas, NumPy, NLTK, Matplotlib, and Jupyter Notebook.

## Getting Started

Clone the repository, install the common dependencies with pip install numpy pandas scikit-learn matplotlib nltk jupyter, then launch Jupyter Notebook and open any notebook inside the topic folders to run it.

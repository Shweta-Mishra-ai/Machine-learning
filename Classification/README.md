## Classification

Implementations of classification algorithms for supervised learning tasks.

## Contents

Naive Bayes: Gaussian and Bernoulli variants (gaussian_naive_bayes.txt, BERNOULLI_NB.txt, BERNOULLI NAIVE BAYES.txt). Multinomial Naive Bayes for text categorization (multi_nomial.txt, NEWS CATEGORIZATION IN MULTINOMIAL NB.txt). K-Nearest Neighbors (knn.txt). Decision Tree Classifier demo (Decision_Tree_Classifier_Demo.ipynb). Naive Bayes built from scratch on the Iris dataset (IRIS_SCRATCH.txt). Digit recognition practice (DIGIT_RECOGNIZE.txt). Loan approval classification practice (loan.txt, loan_repayment_details). The Misc-Practice folder holds smaller one-off classification snippets.

## Datasets used

diabetes.csv, breast_cancer_updated.xlsx, spam.csv, Weather_Dataset_for_GNB.xlsx, weather_datatset_for_NB.csv, PIMA_DESCR.txt

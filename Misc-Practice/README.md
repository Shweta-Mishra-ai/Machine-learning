## Misc Practice

General Python practice scripts not tied to a specific ML topic: loops, recursion, list and string handling, small games, and utility scripts written while learning.

## Contents

class.txt covers OOP practice. fibonacci.txt generates a Fibonacci sequence. lst.txt practices list insert, delete, and update operations. marks.txt generates random numbers for practice. min_dis.txt finds a minimum distance value. my_email.txt sends an email via smtplib (practice script with placeholder credentials). my_work.txt checks for prime, pronic, and Armstrong numbers. neg_pow.txt handles negative exponents. russionrolled.txt combines text-to-speech with a random selection game. sannekladder.txt implements Snakes and Ladders game logic. sec_max_num.txt and secmax.txt find the second largest number using different approaches. sec_min_max.txt finds the second minimum and maximum values. str.txt sorts a string as a set. IPL+Points+Table+Dataframe.ipynb is a pandas practice notebook building an IPL points table. Intruder Alert.mp3 is an audio clip used by one of the practice scripts.
